The scattering matrix data is provided for shape A, for wvl=1.064μm

----------------------------------
Scattering matrix files:
----------------------------------
Each scattering matrix element (in μm2) is provided in a separate file (e.g. S11, S12 etc), for (columns) the scattering angles provided in "scattering angles" file and for (rows) the volume-equivalent size parameters provided in "size parameter" file 

----------------------------------
LR and PLDR files:
----------------------------------
The LR and PLDR are provided for the different volume-equivalent size parameters provided in "size parameter" file
